生成一个账号

填好后点「生成」，把下面的 JSON 块粘进 users.json 的 users 数组里，提交到 GitHub 即可。密码本身不会被保存，只保存哈希值。


        
      

    


    

      
怎么用

      
    
        
  1. 每个员工填一次，点「生成」，复制 JSON 条目。
    2. 
        
  2. 打开 GitHub 仓库里的 users.json，把条目加进 users 数组（条目之间用逗号隔开）。
    3. 
        
  3. 提交（Commit）。GitHub raw 缓存约几分钟后，登录页面即可用新账号。
    4. 
        
  4. 改密码＝重新生成同一用户名的条目，替换旧的即可。删除账号＝删掉对应条目。
    5.
